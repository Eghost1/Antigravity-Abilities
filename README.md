# Antigravity Skills Repository

> **Firma de autoría**: Repositorio y Habilidades creadas por Diego Bravo (<https://github.com/Eghost1>).

## Descripción

Este repositorio contiene una colección de habilidades (skills) avanzadas diseñadas para extender las capacidades cognitivas y operativas del agente **Antigravity**. Cada habilidad está configurada como un paquete de conocimiento reutilizable que encapsula instrucciones específicas, mejores prácticas, guardrails y formatos de salida para dominios altamente especializados (Frontend, Backend, Diseño, SEO, Arquitectura de Sistemas, redacción, etc.).

## 📂 Estructura de una Habilidad

Cada habilidad dentro de `./.agent/skills/` sigue una convención estricta:

1. Un archivo `SKILL.md` principal que modela e instruye el comportamiento y *guidelines* de la IA.
2. Un directorio asociado a la habilidad, pudiendo integrar `/scripts`, `/examples` o `/resources` (opcional).
3. Un archivo `README.md` autogenerado con una guía práctica humana que explica la capacidad adquirida.

> ℹ️ *Aviso automatizado: Cualquier nueva habilidad añadida a través de la herramienta o habilidad `creador-de-habilidades` se indexará automáticamente y dinámicamente en este documento general de Registro.*
