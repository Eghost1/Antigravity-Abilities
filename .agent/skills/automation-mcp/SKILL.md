# Automation & Marketing Tools Skill

Optimiza los flujos de trabajo de comunicación y creación de contenido técnico.

## Capacidades

- **Headless Gmail**: Leer, buscar, enviar y organizar correos sin interfaz gráfica. Ideal para automatizar seguimientos o alertas de sistema.
- **OSP Marketing Tools**:
  - **Product Value Map**: Generar propuestas de valor, personas y declaraciones de posicionamiento.
  - **Meta Info**: Optimizar meta-títulos y descripciones para SEO.
  - **Content Codes**: Revisar contenido técnico bajo estándares profesionales.

## Instrucciones de Uso

1. **Email Workflow**: Automatiza el envío de reportes de SEO generados por la habilidad `google-marketing-mcp`.
2. **Content Lifecycle**: Primero crea el contenido, luego usa OSP para optimizar su SEO y finalmente prográmalo o envíalo por email.

## Ejemplo de Prompt

"Envía un correo a <diego@ejemplo.com> con el resumen del último post. Antes de enviarlo, usa OSP para sugerirme 3 meta-descripciones optimizadas para SEO."
