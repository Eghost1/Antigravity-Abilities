# PPT-pro Skill

Transforma ideas y datos en presentaciones visuales de alto impacto.

## Capacidades

- **Diseño de Diapositivas**: Creación de layouts para portadas, secciones, contenido y cierre.
- **Visualización de Datos**: Integración de tablas y gráficos que respeten la paleta de colores del proyecto.
- **Narrativa Visual**: Estructuración del contenido para que sea legible y atractivo.
- **Multimedia**: Inserción de imágenes y formas geométricas personalizadas.

## Alineación Visual

No generar presentaciones con plantillas genéricas:

1. Analiza el `README.md` y el código para entender el tono (formal, técnico, moderno).
2. Extrae colores primarios y secundarios de los estilos del proyecto.
3. Si no encuentras activos visuales, pide al usuario el logo y la paleta deseada.

## Ejemplo de Prompt

"Prepara una presentación de 5 slides para inversionistas sobre el nuevo módulo de AI. Usa fondos minimalistas y los colores neón que estamos usando en la interfaz del dashboard."
