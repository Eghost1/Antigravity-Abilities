# PDF-pro Skill

Generación de PDFs de alta calidad para reportes técnicos, manuales y facturas.

## Capacidades

- **Layout Avanzado**: Control preciso de márgenes, paginación, encabezados y pies de página.
- **Integración de Medios**: Inserción de imágenes, logos, códigos QR y gráficos vectoriales (SVG).
- **Estilización**: Uso de tipografías personalizadas y paletas de colores extraídas del contexto del proyecto.
- **Interactividad**: Creación de enlaces internos, marcadores (bookmarks) y metadatos del documento.

## Alineación Visual

Es imperativo que el PDF se sienta como una extensión del software o marca:

1. Revisa los archivos `.css` del frontend para replicar degradados, sombras y fuentes.
2. Integra el logo del proyecto si está disponible en el repositorio.
3. Consulta al usuario si hay un "manual de marca" o lineamientos específicos de diseño.

## Ejemplo de Prompt

"Crea un PDF técnico que explique el funcionamiento del backend. Incluye diagramas de arquitectura (si los encuentras en el repo) y usa la misma estética de 'modo oscuro' que tiene la landing page."
