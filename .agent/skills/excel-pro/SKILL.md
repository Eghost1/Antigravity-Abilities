# Excel-pro Skill

Permite generar hojas de cálculo profesionales con alta precisión técnica y estética.

## Capacidades

- **Cálculos y Fórmulas**: Implementación de fórmulas básicas (SUM, AVG) y avanzadas (VLOOKUP, IF anidados, cálculos matriciales).
- **Estilo Visual**: Aplicación de colores de celda, bordes, fuentes y formatos condicionales alineados con la identidad visual del proyecto.
- **Integración de Datos**: Capacidad para volcar grandes conjuntos de datos y estructurarlos en tablas dinámicas o rangos con nombre.
- **Gráficos**: Generación de representaciones visuales de los datos dentro del mismo archivo.

## Alineación Visual

Antes de generar el documento:

1. Escanea el codebase para identificar colores (HEX/RGB) y estilos predominantes.
2. Si el proyecto tiene un `brand.json` o similar, úsalo para definir los colores de las cabeceras y acentos.
3. Si no hay información visual clara, consulta al usuario por su paleta de colores o logos preferidos.

## Ejemplo de Prompt

"Genera un reporte de ventas mensual en Excel. Usa fórmulas para calcular el margen de beneficio y aplica los colores corporativos (azul oscuro y oro) que viste en el branding del sitio web."
